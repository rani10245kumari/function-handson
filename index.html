Q.1  Write the code to acess an element which is having id as "text" ?

Ans. To access an element with the id "text" using JavaScript, you can use the getElementById() method.
For example- 

        var element = document.getElementById("text");

Q.2   Write the code to acess element which is having tag as "h1"?

Ans. To access an element with tag "h1" using JavaScript, you can use the getElementByTagName() method.

            var element = document.getElementByTagName("h1");

Q.3   Write the code to acess element which is having class as "box"?

Ans. To access an element with the class "box" using JavaScript, you can use the getElementByClassName() method.

            var element = document.getElementByClassName("box");

Q.8  What's the difference between Window vs Document ?

Ans- In JavaScript, window and document are both global objects but they have some difference

window: Document:
The window object represents the browser window or tab that contains the web page. The document object represents the DOM (Document Object Model) for a web page.
It serves as the global object in the browser's JavaScript environment. It is a property of the window object and provides an interface to access and manipulate the elements and content within the web page.
The window object provides access to the overall browser environment the document object allows manipulation and interaction with the specific HTML document loaded in the browser.
